1. I cannot personally solve the problem without external help (i still can look at the hints taht is provided by the problem itself)
2. I cannot solve the problem in this timeframe (30-45 minutes) for easy problem, (45-60 minutes) for medium problem (60-90 minutes) for hard problem
3. If i cannot solve the problem in the first try, it is prohibited that i immediately see the solutions and then i also failed when attemptuing it for the next day

This repo was created because i need to personally assess myself and reflect on what is my weakness when solving leetcode problems
